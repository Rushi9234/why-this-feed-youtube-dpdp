# 🎯 Why This Feed?

> A Product Management case study redesigning YouTube's AI recommendation transparency and consent experience under India's Digital Personal Data Protection (DPDP) Act, 2023.

## 📌 Overview

This project reimagines YouTube's recommendation transparency by helping users understand **why content is recommended before making privacy decisions**.

Instead of treating privacy as a legal workflow, the proposal transforms it into a trust-building product experience.

---

## 🚀 Problem

Users enjoy personalized recommendations but rarely understand:

- Why a video is recommended
- What personal data is being used
- How changing privacy settings affects recommendations

This results in consent that is technically valid but often not meaningfully informed.

---

## 💡 Solution

**Why This Feed?**

A contextual recommendation transparency experience built on three principles:

- Explain
- Predict
- Decide

Users can:

- Understand recommendation signals
- Preview changes before updating privacy preferences
- Manage purpose-wise consent
- Protect children's data through verified parental consent
